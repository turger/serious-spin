# Baron Chill W Cloud

Rather cool word cloud article trend visualizer.

## Setup

Add firebase:  
`npm install -g firebase-tools`

Login to firebase:  
`firebase login`

add .env file to project root:  
REACT_APP_APIKEY=  
REACT_APP_AUTHDOMAIN=  
REACT_APP_DATABASEURL=  
REACT_APP_PROJECTID=  
REACT_APP_STORAGEBUCKET=  
REACT_APP_MESSAGINGSENDERID=  

`yarn`

`yarn start`

## Deploy to firebase

`yarn build-fb`
